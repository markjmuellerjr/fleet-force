# FleetForce

FleetForce is a Next.js application designed to manage fleet operations efficiently.

## Features

- User authentication with NextAuth
- MongoDB Atlas integration
- Data seeding with faker.js
- Responsive UI with Tailwind CSS

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
